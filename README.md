# cpk
intended simple package manager for C languae

# installation steps

```sh
./cpk compile
```

```sh
sudo cp bin/cpk /usr/bin
```
## Usage ->

To make a simple cpk project use
```sh
$ cpk mksol <projectname>
```
To compile the said project use
  ```sh
$ cpk compile
```
## Note it only wroks on linux
